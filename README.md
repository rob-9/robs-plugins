# robs-plugins

obsidian plugins mainly for coding workflows

| Plugin | Description |
|--------|-------------|
| [code-styler-default-lang](./code-styler-default-lang) | Per-file default language for inline code syntax highlighting via frontmatter |
| [source-bridge](./source-bridge) | Clickable local file paths with expandable syntax-highlighted code previews |
| [complexity-sidebar](./complexity-sidebar) | Extracts Big-O annotations from code blocks and displays them in a sidebar panel |
| [spaced-rep-code](./spaced-rep-code) | Code-aware flashcards with cloze deletion, output prediction, and complexity quizzes |
| [dataview-lite](./dataview-lite) | Simple query language for notes — a beginner-friendly alternative to Dataview |
